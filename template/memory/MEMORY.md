# Memory Index

Долгая типизированная память. Каждая память — отдельный `.md` в `memory/` с frontmatter:

```markdown
---
name: <kebab-slug>
description: <одна строка — по ней решаю релевантность при recall>
metadata:
  type: user | feedback | project | reference
---

<факт; для feedback/project добавь **Why:** и **How to apply:**; связи через [[other-name]]>
```

Типы: `user` — кто владелец; `feedback` — как мне работать (с «почему»); `project` — текущая работа/цели/ограничения, не выводимые из кода; `reference` — внешние ресурсы (URL, дашборды, тикеты).

Правила: перед записью проверь дубликат — обновляй существующий файл, не плоди копии; удаляй неверное; не сохраняй то, что и так в коде/git/CLAUDE.md.
